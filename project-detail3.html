Descrição do Projeto

Uma aplicação para consulta de endereço utilizando o ceep via api, a aplicação será distribuida em multiplos containers usando docker, a aplicação é constituida por 5 containers.

Descrição dos Containers

  • Container Ator: Uma aplicação web implementada com flask, por onde o usuario pode enviar seu cep, apartir de uma requisição recebida envia a requisição para o container produtor.
  • Container Produtor: Uma aplicação implementada com flask, que recebe requisições do Container Ator e envia para uma fila rabbitmq.
  • Container rabbitmq: Container que roda um servidor rabbitmq responsavel por armazenar as mensagens enviadas pelo Container Produtor.
  • Container Consulmidor: Container responsavel por receber as mensagens da fila do Container rabbitmq e converter os dados da requisições da fila, acessando uma api externa de endereções que retorna as informações de localização dado um cep, com os novos dados obtidos, os dados são enviados para o container mysql.
  • Container Mysql: Container que roda um servidor mysql, responsavel por receber os dados do Container Consulmidor e armazena-los no banco de dados.

Tecnologias Utilizadas
  • Docker
  • Docker Compose
  • Python
  • flask
  • Fila de Menssageria rabbitmq
  • Banco dados Mysql
